TICKOLO - An Android App for Fan to Fan Ticket Exchange

Date: Nov 2013
Worked with Android OS, Java, PHP, SQL

Role: Software, UI Design
Team mates: None


WHAT IS IT?

Tickolo is a prototype for a ticket exchange platform in which users can buy and sell spare tickets at less than face value. It was made as an assignment for the course Software Structures for User Interfaces. This app currently works on the android platform and a beta version will be launched by the end of May 2014.



WHY WAS IT MADE?

Tickolo is unique in its fan-based ticket-exchange system where it places a restriction on the price of the ticket. Tickets can never be sold at prices higher than the face value thereby discouraging scalping and promoting good deals for people who want last minute tickets.



HOW WAS IT DONE?

Tickolo involved both client and server side programming. The backend (which is hosted on Openshift), is programmed using PHP and SQL. The android client facing application was programmed using Java.
