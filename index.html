Tango is a prototype for a news website created for HackCMU. It presents news in different layouts based on the distance the viewer is from the screen. It uses a web camera to estimate the distance of the user's face from the screen. Based on the distance it has three different layouts (detail view, browse view and full screen video) Tango was awarded the 'Most Innovative Project' at HackCMU 2013.

SortAway - An exciting new way to search SortAway is a website which aims to help users make quicker and better consumer decisions. It's currently targeted at the bludgeoning phone market in India and automatically chooses the model that best fits the user's requirements. We expect a beta version of the site to released by march 2014.

I interned Oracle Financial Services Software Limited (OFSSL) where I enhanced, supported and devoloped tools used by the development and the test team. I used ASP.NET to design the user interface and a combination of VB.NET, javascript and SQL to process user data.


Tickolo is a prototype for a ticket exchange platform in which users can buy and sell spare tickets at less than face value. This system is ideal for people looking for last minute ticket deals. This app currently works on the android platform and a beta version will be launched by the end of January 2014.
